# General Agent Prompt Drafts

This folder is for our custom prompt drafts that are intended to make Codex behave more like a general-purpose agent rather than a coding-only agent.

Suggested workflow:
- Keep each substantial revision in its own file, such as `system_prompt_v1.md`, `system_prompt_v2.md`, and so on.
- Use this folder for authored drafts only.
- Keep the extracted upstream prompts in `prompts/` unchanged for reference.

Current files:
- `system_prompt_v1.md`: first working draft for a general-purpose Codex system prompt.
